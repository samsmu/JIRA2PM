JIRA2PM Help

Connection settings

Connection settings consist of 3 fields:

  • Username - username which you use in JIRA
  • Password - password for your account
  • Base URL - base url of your JIRA server. In case of cloud JIRA hosting it will be looks like https://example.atlassian.net/
After filling all fields, press connect to connect to JIRA. Button should be changed to the success message after connecting.

JQL Query

  • JQL Query - A query which you want to fetch. You can create it in JIRA and then copy to this field
  • Fields to Show - Fields which will be displayed in table
Important Note: At this moment only GiHub control version is supported.

Display Options

  • Fields Displayed as Hyperlinks - selected fields will be displayed as hyperlinks to issues or JQL queries
  • Date Format - Format to display dates. Y - year, M - month, D - day, h - hour, m - minute, s - second
  • Estimation Unit - Unit in which estimations will be displayed
  • Sort by parents - Sort issues by epics and subtasks. Epics have additional formatting

Setup Storage

  • Refresh every... - If this option has been selected - the add-on should check was the last row inserted in the table before the selected time interval. If it was, the old information will be updated

Get Result

  • Send Request - Sends request to JIRA with selected options and inserts received data into active sheet
  • Clean Sheet and Send Request - Additionaly cleans sheet before inserting data from request
  • Send Request and Insert Data into New Sheet - Inserts data from request to newly created sheet
